# Flipkart Clone - Frontend

This project is a frontend clone of Flipkart, an e-commerce platform, built with HTML, CSS, and JavaScript.

## Features:
- Product Listing
- Search Bar
- Shopping Cart
- Responsive Design

## Technologies Used:
- HTML
- CSS
- JavaScript

## Setup Instructions:
1. Clone the repository.
2. Open `index.html` in your browser.
